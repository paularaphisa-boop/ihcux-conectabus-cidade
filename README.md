Contexto de Uso O aplicativo é usado por pessoas que precisam chegar rápido ao destino, como estudantes e trabalhadores correndo para não perder o ônibus ou chegar no horário.

Decisões de UX As informações mais importantes, como rota, tempo de chegada, saldo e lotação do veículo, foram destacadas para facilitar a visualização rápida e evitar distrações.

Acessibilidade O protótipo mostra informações de acessibilidade do ônibus, como rampa para cadeirantes e espaço reservado. Além disso, possui botões grandes e navegação simples par
